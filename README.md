Feature List to complete:

1. Manage customer data with the dashboard generate invoice system
- Cloud based dashboard
a lightweight dashboard based on a cloud server that can be accessed through various devices without the need to download an application

- Data Transfer
Uploading customer data from Excel and CSV files into the Generate Invoice System dashboard in one process

- Data Management
make it easier for Bill Issuers to manage customer data with various supporting features

2. Automated billing and transaction status tracking
- Bulk billing
Automate billing and delivery periods in bulk to customers (Sending invoices via Payment Gateway)

- Payment Reminder
Sending payment reminders to multiple customers via email

- Real-time and comprehensive data
Payment status is presented in real-time and comprehensive transaction reports

3. Expansion of payment options to multiple payment channels and partners
